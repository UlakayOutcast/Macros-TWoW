Цвета текста в чате (макрос. по прозьбе многих))
Думаю что большинство знает как создавать макросы в ВоВ, но все же

Standart(это тот макрос который возвращает все в нормальное состояние) (думаю его нужно первым создать)) ):
/script if (not scm) then scm = SendChatMessage; end; function SendChatMessage(msg, type, lang, chan) scm(msg, type, lang, chan); end;

Теперь на цвета. Как понимаете макрос один и при изменении определённого параметра определяется цвет текста. Вот макрос:

/script if (not scm) then scm = SendChatMessage; end; function SendChatMessage(msg, type, lang, chan) scm("\ХХХХХХХХХХХХ\124Hitem: 19:0:0:0:0:0:0:0\124h"..msg.."\124h\124r", type, lang, chan); end;

где ХХХХХХХХХХХХ - и есть параметр определяющий цвет) :
124cffffffff - белый
124caa88aa88 - серый
124c00000000 - чёрный
124cff00ff99 - бирюзовый
124cFFC2C050 - жёлтый
124caa55aa55 - зелёный_1
124cff00ff00 - зелёный_2(этот кажись светлее)
124cFFF00000 - красный
124c00FF23CC - розовый

prof Некоторые замечания: цвет изменяется в любом чате; его видят таким все игроки, а не только вы; при включённом макросе не работает команда ".save"
